How to read this page

Theme changes color tokens. Appearance changes structural feel (radius, density, and platform-like spacing cues). Active context changes interaction emphasis for control groups.

Try setting one side to attribute scope and the other to class scope. You should get the same visual result, which proves both APIs are equivalent.
